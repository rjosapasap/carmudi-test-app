# Carmudi Test App

## Implemented Goals

1. Load data from web service
2. UI adaptive to orientation
3. Product list with details and image
4. Pull-to-refresh
5. Infinite Scrolling
6. UI for sorting
7. Cache Images (via Glide)
8. Implement using MVP pattern
